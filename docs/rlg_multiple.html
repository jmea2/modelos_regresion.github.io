Modelo propuesto para el estudio de enfermedad coronaria

mod_base <- glm(chd69 ~ age + chol + sbp + bmi + smoke,
data = wcgs, family = binomial)
extract_eq(mod_base)

\[ \log\left[ \frac { P( \operatorname{chd69} = \operatorname{1} ) }{ 1 - P( \operatorname{chd69} = \operatorname{1} ) } \right] = \alpha + \beta_{1}(\operatorname{age}) + \beta_{2}(\operatorname{chol}) + \beta_{3}(\operatorname{sbp}) + \beta_{4}(\operatorname{bmi}) + \beta_{5}(\operatorname{smoke}_{\operatorname{Yes}}) \]

Evaluación de interacción

Probaremos interacciones candidatas y seleccionaremos la más prometedora (por ejemplo, chol * bmi). Se ilustran dos vías: prueba de Wald (coeficiente) y LRT (comparación de modelos).

# Candidata: interacción entre el colesterol y el IMC

mod_int <- glm(chd69 ~ age + chol + sbp + bmi + smoke + chol:bmi,
data = wcgs, family = binomial)
