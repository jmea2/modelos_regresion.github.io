Interacción

Se quiere evaluar en el modelo de efectos principales del programa de actividad fisica sobre la glucosa la posibilidad de una heterogeneidad de efectos mediada por el indice de masa corporal. Para este proposito se ajusto un modelo con un termino de interacción, entre BMI y el programa de ejercicio:

  • Escribe la ecuación del modelo con el termino de interacción:
mod2 <- lm(formula = glucose ~ exercise + BMI + exercise:BMI, data = hers)
